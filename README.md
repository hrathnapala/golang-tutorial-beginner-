# Go Lang Tutorial

* Programming Language Developed at Google in 2007.
* Open - Sourced in 2009.

## Go Use Cases

* Go was designed to run on multiple cores and built to support concurrency
* Concurrency in go is cheap and easy
    * So that can write applications which is very performant
    * Running on scaled, distributed systems

## Characteristics

* Simple Syntax
* Fast build time, start up and run
* Requires fewer resources
